# Employee Management System

A role-based Employee Management System built using React.js and Tailwind CSS.

## Features
- Admin Login
- Employee Login
- Create and Assign Tasks
- Task Status Management
- Dashboard for Admin and Employees
- React Context API
- Local Storage for Data Persistence

## Tech Stack
- React.js
- JavaScript
- Tailwind CSS
- Context API
- Local Storage

## Installation

```bash
npm install
npm run dev
